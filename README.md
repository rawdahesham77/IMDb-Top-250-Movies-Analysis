# IMDb-Top-250-Movies-Analysis

## 📌 Project Overview

This project analyzes the IMDb Top 250 movies dataset using Power BI to uncover patterns, trends, and key insights about highly rated movies.

## 🛠️ Tools Used

* Power BI
* Excel

## 📊 Key KPIs

* Total Movies: 250
* Average Duration: 142 minutes
* Total Reviewers: 185M+
* Highest Rating: 9.3
* Lowest Rating: 8.0

## 🔍 Analysis Questions

* Which movies have the highest ratings?
* Does movie duration affect rating?
* Which decade has the most top-rated movies?
* How do reviewers impact ratings?

## 📈 Key Insights

* Top-rated movies have consistently high ratings (8.0+).
* Movies tend to be longer in duration (~142 minutes).
* High number of reviewers indicates strong audience engagement.
* Ratings differences are small, showing overall high quality.
